### Yet Another React Redux Webpack Starter Kit

* [x] Dev and prod modes
* [ ] Modern ESLint
* [ ] Modern Babel
* [ ] React Storybook
* [ ] Redux Saga

#### Pre-requested
```
npm install
```

#### Dev mode
```
npm run dev
```

#### Prod mode
```
npm run dev
```

#### Bootstrap
```
git clone git@github.com:soylent-grin/react-redux-webpack-starter-kit.git && \
    cd react-redux-webpack-starter-kit && \
    npm i && \
    npm run dev
```
